# The RPR Lab AWS User's Group

The RPR Lab AWS environment provides shared compute and storage for research. Use these docs to access AWS, learn how to launch reproducible analysis environments, manage working storage, and preserve project data.

## Getting help

!!! info "Use the AWS User's Group channel"

    To ask questions, request access, or report a problem, contact an Owner or Administrator, or send a message to the AWS User's Group channel in Slack.

## Roles

### AWS Owners

Owners have full access to the AWS organization, including the ability to manage accounts, billing, recovery access, governance, and highly privileged changes. All Owners are also Administrators.

### AWS Administrators

Administrators have the ability to manage identities, EC2, storage, security, AMIs, launch templates, and resource lifecycles.

### AWS Researchers

Researchers have the ability to use all resources, including launching and managing EC2 instances, and storing and accessing project data in S3.

!!! failure "Keep the AWS environment secure"

    Owners and Administrators will never ask you to send passwords, access keys, MFA codes, private keys, or anything secret through email or other digital channels. If something seems sketchy, speak to one of us in person.
